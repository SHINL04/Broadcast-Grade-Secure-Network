\# Broadcast-Grade Secure Network



\## Project Overview



This project simulates a broadcast/media company network using Cisco Packet Tracer.



Features implemented:



\* VLAN Segmentation

\* Router-on-a-Stick

\* OSPF Dynamic Routing

\* Inter-VLAN Communication

\* Remote Office Connectivity

\* Enterprise IP Addressing

\* End-to-End Network Verification
